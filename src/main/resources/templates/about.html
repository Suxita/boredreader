How It Works

BoredReader combines several technologies:

  • PDF.js - For rendering books directly in your browser
  • Spring Boot - backend
  • TinyLlama -AI model only for initial showcasing

When you ask a question, system analyzes the current page you're reading and provides context-aware responses
